# 二，第二个模块

该模块是滑流动量理论，以其中的一个模块为例

首先计算体轴系下的滑流区合空速



function \[u\_total,v\_total,w\_total\] = get\_uvw\_total\(n\_s,CT,Prop\_angle,Vb\_air\)

u\_air = Vb\_air\(1\);

v\_air = Vb\_air\(2\);

w\_air = Vb\_air\(3\);

%V = u\_air\*cos\(Prop\_angle\)\-w\_air\*sin\(Prop\_angle\);

%Thrust = max\(0,THRUST\);

D\_out = 0\.2032;

a0 = 1\.59/2;  %1\.33

V\_i = a0\*n\_s\*D\_out\*sqrt\(CT\);

u\_total = u\_air;%\+V\_i\*cos\(Prop\_angle\);

v\_total = v\_air;

w\_total = w\_air;%\-V\_i\*sin\(Prop\_angle\);



之后我们可以获得Vb = \[u\_total, v\_total, w\_total\];

通过把这个Vb输入simulink的Incidence, Sideslip,\& Airspeed 模块获得 **alpha\_slip1**, beta, V,我们只取用alpha\_slip1。

另外把这个Vb输入simulink的Dynamic Pressure模块获得DP\_slip1\.



通过一个MATLAB function获得f\_f1,这个f\_f1如下：

function f\_f = fcn\(u\_e,Vb\_air,alpha\_slip\)

a0 = 0\.3/\(0\.5\*0\.2032\);

a1 = 0\.5;

u\_inf = Vb\_air\(1\);

lamda = \(u\_e\.^2\-u\_inf\.^2\)\./\(u\_e\.^2\+u\_inf\.^2\);

sigma = 1 \+ lamda\./\(1\+4\*a1^2/a0^2\+4\*a1/a0*sin\(alpha\_slip\)\) \.\.\.*

*\+ lamda\./\(1\+4*a1^2/a0^2\-4\*a1/a0\*sin\(alpha\_slip\)\) \.\.\.

\+ lamda\.^2\./\(1\+4/a0^2\+4/a0*sin\(alpha\_slip\)\) \.\.\.*

*\+ lamda\.^2\./\(1\+4/a0^2\-4/a0*sin\(alpha\_slip\)\);

f\_f = 1\./sigma;





# 三，第三个部分

从外界环境中取出alpha,记作alpha\_free；取出动压DynamicPressure，记作DP\_free。

我们的alpha做如下处理

存在一个常数alpha\_slip\_switch = 1\.

然后alpha\_slip1,alpha\_slip\_switch,alpha\_free 输入 simulink的switch模块，判别条件是 \> 0\.5。输出的就是第一个模块使用的alpha\_slip1。

滑流区动压也是和迎角一样处理的，存在常数DP\_slip\_switch = 1，之后做判别获得DP进入第一模块。

滑流区面积S\_slip1\~8在原本的General结构体中定义。

在这里有一个多余的接口，现在的f\_f用的都是1\.
