# 滑流建模说明

本文说明 `matlab_model/tandem_aero_fm.m` 中 8 个主桨滑流区气动力的建模方式、参数开关、验证结果，以及一个静止地面大迎角工况下的滑流升力估算。

## 1. 建模目标

当前模型把机翼分成两部分：

1. `S_free`：不在主桨滑流覆盖区内的自由来流翼面。
2. `S_slip(1:8)`：8 个主桨对应的滑流覆盖翼面。

滑流模型只计算 `S_slip` 区域的气动力增量，不计算滑流气动力矩。也就是说：

```matlab
f_a = F_elevon + F_sfree + F_sw + F_slip;
m_a = M_elevon + M_sfree + M_sw;
```

其中 `F_slip` 是 8 个滑流面板的合力，`m_a` 中暂时不包含滑流力矩。

## 2. 参数开关

滑流相关开关在 `matlab_model/init_param_zx.m` 中定义：

```matlab
param.slipstream_enable = true;        % 是否考虑 8 个主桨滑流区诱导速度
param.slipstream_ff_enable = true;     % 是否使用滑流区 f_f 修正系数；false 时 f_f=1
param.alpha_slip_switch = 1;           % 是否使用滑流局部迎角
param.DP_slip_switch = 1;              % 是否使用滑流局部动压
```

推荐默认设置：

```matlab
param.slipstream_enable = true;
param.slipstream_ff_enable = true;
```

如果需要做对比：

```matlab
param.slipstream_enable = false;       % 不加桨诱导速度，但仍保留 S_slip 自由来流气动力
param.slipstream_ff_enable = false;    % 使用滑流诱导速度，但强制 f_f=1
```

注意：`slipstream_enable=false` 不是删除 `S_slip` 翼面，而是让 `S_slip` 使用自由来流计算。这样可以避免关滑流时少算一部分机翼面积。

## 3. 单个滑流面板计算流程

对第 `i` 个主桨滑流面板：

### 3.1 旋翼状态

先用当前油门和自由来流状态调用主桨模型：

```matlab
[~, ~, n_rps_i, CT_i] = tandem_rotor_thrust(dt_i, alpha, beta, Va, ...
                                            param.prop_spin(i), param, ...
                                            param.prop_angle(i));
```

得到：

- `n_rps_i`：第 `i` 个桨转速，单位 rps。
- `CT_i`：推力系数。

### 3.2 诱导速度

按滑流动量理论估算诱导速度：

```matlab
Vi = a0 * n_rps_i * D * sqrt(CT_i);
```

其中：

```matlab
D = param.prop_D;      % 默认 0.2032 m
a0 = 1.59 / 2;
```

局部滑流速度在机体系下写成：

```matlab
u_local = u_air + Vi * cos(prop_angle);
v_local = v_air;
w_local = w_air - Vi * sin(prop_angle);
```

当前模型中 `param.prop_angle = 0`，因此滑流主要增加机体系 `x` 向速度。

### 3.3 局部迎角和动压

局部速度：

```matlab
Va_local_b = [u_local; v_local; w_local];
```

局部迎角和侧滑角：

```matlab
alpha_slip = atan2(w_local, u_local);
beta_slip  = atan2(v_local, hypot(u_local, w_local));
```

局部动压：

```matlab
qbar_slip = 0.5 * rho * dot(Va_local_b, Va_local_b);
```

### 3.4 气动系数

用局部迎角查表：

```matlab
CD_slip = aero_cd(param, alpha_slip);
CL_slip = aero_cl(param, alpha_slip);
CY_slip = 0;
```

再转成机体系气动力系数：

```matlab
CAx = -CD*cos(alpha)*cos(beta) - CY*cos(alpha)*sin(beta) + CL*sin(alpha);
CAy = -CD*sin(beta) + CY*cos(beta);
CAz = -CD*sin(alpha)*cos(beta) - CY*sin(alpha)*sin(beta) - CL*cos(alpha);
```

### 3.5 `f_f` 修正

默认使用原始文档中的 `f_f`：

```matlab
lambda = (u_e^2 - u_inf^2) / (u_e^2 + u_inf^2);

sigma = 1 ...
    + lambda   / (1 + 4*a1^2/a0^2 + 4*a1/a0*sin(alpha_slip)) ...
    + lambda   / (1 + 4*a1^2/a0^2 - 4*a1/a0*sin(alpha_slip)) ...
    + lambda^2 / (1 + 4/a0^2    + 4/a0*sin(alpha_slip)) ...
    + lambda^2 / (1 + 4/a0^2    - 4/a0*sin(alpha_slip));

f_f = 1 / sigma;
```

其中当前取值：

```matlab
a0 = 0.3 / (0.5 * 0.2032);
a1 = 0.5;
```

最终第 `i` 个滑流面板气动力为：

```matlab
F_slip_i = f_f * qbar_slip * S_slip(i) * CA_slip;
```

8 个面板求和得到：

```matlab
F_slip = sum(F_slip_i);
```

## 4. 无来流静止工况处理

为了支持“地面静止开桨”的滑流计算，`tandem_aero_fm.m` 现在不再在 `Va=0` 时直接返回零气动力。

当前处理方式：

1. 若 `Va=0`，自由来流气动力、舵面气动力和动导数气动力为零。
2. 若 `slipstream_enable=true` 且主桨有油门，`S_slip` 面板仍按桨诱导速度计算局部动压和局部迎角。
3. 若 `slipstream_enable=false`，`S_slip` 面板只使用自由来流；在 `Va=0` 时自然为零。

这样可以计算悬停、地面支撑、大迎角起飞等低速滑流工况。

## 5. 滑流效果对比

参考工况：

```matlab
Ve = [10; 0; 1];       % 地固系速度
wind = [0; 0; 0];
dt1...dt8 = 0.5;
delta_aeL = 0;
delta_aeR = 0;
```

气动力结果：

```text
无滑流       F_off = [-1.21258, 0, -34.1058] N
滑流 f_f=1   F_ff1 = [-2.15214, 0, -44.2954] N
完整 f_f     F_on  = [-1.57431, 0, -33.9218] N
```

换成机体系向上气动力大小：

```text
无滑流       34.1058 N
滑流 f_f=1   44.2954 N，约为无滑流的 1.299 倍
完整 f_f     33.9218 N，约为无滑流的 0.995 倍
```

这说明：

1. 仅考虑诱导速度和局部动压时，滑流增升比较明显。
2. 使用原始 `f_f` 后，`f_f` 对滑流气动力有很强削弱作用。
3. 在该参考工况下，完整滑流模型的净增升并不明显。

## 6. 指定地面工况估算

用户指定工况：

```text
无人机俯仰角：40 deg，机体支在地面上
地速 Ve：0 m/s
风速 wind：0 m/s
主桨油门 dt1...dt8：0.95
舵面：0
只考虑滑流气动力，不包含旋翼直接推力
```

坐标定义：

- 机体系气动力 `F_body` 来自 `tandem_aero_fm`。
- 地固系采用 NED，`z` 轴向下。
- 竖直向上升力定义为 `Lift_vertical = -F_earth_z`。

计算得到：

```text
完整 f_f：
F_body = [-0.292895665, 0, -2.16517816] N
F_earth = [-1.61612079, 0, -1.47035299] N
竖直向上滑流升力 Lift_vertical = 1.47035299 N

如果强制 f_f=1：
F_body = [-1.21997512, 0, -9.01844516] N
F_earth = [-6.73149997, 0, -6.12434491] N
竖直向上滑流升力 Lift_vertical = 6.12434491 N
```

因此，在当前完整 `f_f` 模型下，该工况只考虑滑流气动力时，竖直向上的滑流升力约为：

```text
1.47 N
```

该数值不包含主桨直接推力。作为参考，同一工况下单个主桨直接推力约为：

```text
T_single = 6.519 N
```

8 个主桨在机体俯仰 40 deg 时，直接推力的竖直向上分量约为：

```text
8 * T_single * cos(40 deg) = 39.95 N
```

也就是说，在该静止地面工况下，主桨直接推力远大于当前模型计算出的滑流气动升力。

## 7. 合理性判断

当前结果的主要特点是：

1. `f_f=1` 时，滑流诱导速度会带来比较明显的气动升力。
2. 使用原始 `f_f` 后，滑流升力被显著削弱。
3. 静止无来流、油门 0.95、俯仰 40 deg 工况下，完整模型的滑流竖直升力约 `1.47 N`，相对主桨直接推力较小。

因此，如果希望滑流增升在低速起飞中表现得更明显，需要重点复核：

1. `f_f` 公式是否应在静止无来流时直接使用。
2. `u_e` 是否应取局部总速度、轴向诱导速度，还是桨盘出口速度。
3. `CL(alpha=0)` 和滑流区迎角定义是否与 Simulink 原模型完全一致。
4. 是否需要把桨盘安装角、机翼局部安装角或地面支撑姿态引入滑流面板局部迎角。

在未进一步校核原 Simulink 模型或试验数据前，建议保留当前默认 `f_f`，并把 `slipstream_ff_enable=false` 仅作为灵敏度分析开关。
